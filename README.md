# Movie_Library

## To run on local machine
Open your terminal and enter following commands

  git clone https://github.com/kartikbansal/movie_library.git
  cd movie_library
  npm install
  npm start

## To deploy on github pages
Open your terminal and enter following commands

  git clone https://github.com/kartikbansal/movie_library.git
  cd movie_library
  npm install
  npm run build
  git subtree push --prefix dist origin gh-pages
